# 各域名商特色詳述

## GoDaddy

**定位：全球最大域名商，適合初學者**

```
優點：
+ 全球最大，知名度高，遇問題資源多
+ 首年價格非常便宜（促銷吸客）
+ 有中文介面與中文客服
+ 提供 Website Builder、Hosting 一站式服務
+ 支援多種付款方式

缺點：
- 續費價格大幅上漲，首年與續費差距可達 10 倍
- WHOIS 隱私保護需額外付費（每年約 $9.99）
- 介面複雜，常推銷不需要的附加服務
- 整體口碑在技術社群不佳（過度行銷）
```

**最適合：** 第一次買域名、需要中文客服支援、短期試用或不在乎續費成本的用戶

---

## Namecheap

**定位：均衡的中階選擇，開發者首選之一**

```
優點：
+ 免費 WHOIS 隱私保護（終身）
+ 首年有小幅折扣，續費價格合理
+ 介面清晰，操作直覺
+ 提供 Email Forwarding、URL Redirect
+ 在開發者社群口碑良好

缺點：
- 無中文介面
- 首年促銷不如 GoDaddy 大
- 客服回應速度時有不穩定
```

**最適合：** 個人開發者、初中階用戶、需要免費 WHOIS 的用戶

---

## Cloudflare Registrar

**定位：長期持有最划算，整合 Cloudflare 生態系最佳**

```
優點：
+ 以成本價（at-cost）銷售，無任何加價
+ 首年價格 = 續費價格，沒有「首年陷阱」
+ 免費 WHOIS 隱私保護
+ 購買後自動整合 Cloudflare DNS、CDN、SSL
+ 不需另外設定 Nameserver
+ 支援 DNSSEC 一鍵啟用

缺點：
- 必須先有並使用 Cloudflare 帳號
- 不支援部分 ccTLD（.tw、部分亞洲域名）
- 首年不提供折扣（比 GoDaddy 首年貴）
- 不提供 Hosting 等附加服務
```

**最適合：** 已在使用 Cloudflare 的用戶、長期持有域名、不想每年被漲價的用戶

---

## Porkbun

**定位：價格最低，附加服務豐富**

```
優點：
+ 多數 TLD 為市場最低價之一
+ 免費 WHOIS 隱私保護
+ 免費 SSL 憑證（Let's Encrypt 整合）
+ 免費 Email Forwarding、URL Forwarding
+ 介面現代、操作流暢
+ 開發者社群評價高

缺點：
- 無中文介面
- 知名度較低，部分用戶對穩定性存疑
- 客服以 Email/Ticket 為主，無即時聊天
```

**最適合：** 價格敏感用戶、購買多個域名、需要附加免費服務的用戶

---

## Gandi

**定位：歐洲老牌，技術用戶信賴**

```
優點：
+ 透明定價，無隱藏費用，首年＝續費
+ 免費 WHOIS 隱私保護
+ 每個域名附贈 2 個 Email 信箱
+ 自有 Anycast DNS（LiveDNS），速度快
+ 支援大量歐洲 ccTLD
+ 符合 GDPR 規範，資料存於歐洲
+ 以「No Bullshit」理念著稱，不過度行銷

缺點：
- 價格偏高（.com 約 $15/年）
- 無中文介面
- 介面相對老舊
```

**最適合：** 需要歐洲 ccTLD（.fr、.de、.eu）、重視資料隱私（GDPR）、不喜歡被行銷騷擾的用戶

---

## Dynadot

**定位：中文友善，亞洲用戶首選**

```
優點：
+ 提供繁體中文與簡體中文介面
+ 支援支付寶、銀行轉帳等亞洲付款方式
+ 免費 WHOIS 隱私保護
+ 價格合理，無首年陷阱
+ 提供域名市場（二手域名買賣）
+ 有 API 支援

缺點：
- 知名度低於 GoDaddy、Namecheap
- 部分新 TLD 支援較少
- 客服回應時間不穩定
```

**最適合：** 需要中文介面的用戶、台灣 / 亞洲市場、習慣使用亞洲支付工具的用戶

---

## Squarespace Domains（前 Google Domains）

**定位：原 Google Domains 用戶的延續**

```
優點：
+ 承接 Google Domains，原有域名自動遷移
+ 整合 Google Workspace Email
+ 價格透明（$12/年 .com，首年＝續費）
+ 免費 WHOIS 隱私保護
+ 介面簡潔

缺點：
- 主要吸引力來自 Google 品牌，非 Squarespace 本身
- 不如 Porkbun / Cloudflare 便宜
- 部分用戶對 Squarespace 接管後的長期穩定性存疑
- 無中文介面
```

**最適合：** 原 Google Domains 用戶（已自動遷移）、使用 Google Workspace 的用戶

# 選擇策略

## 依使用情境選擇

### 🔰 第一次買域名，不確定要用多久

```
推薦：Namecheap 或 Porkbun

理由：
- 首年價格合理（非陷阱式促銷）
- 免費 WHOIS 隱私
- 介面簡單，資源豐富
- 即使後悔也可轉移域名
```

### 🌩️ 已在使用 Cloudflare（CDN / 防護）

```
推薦：Cloudflare Registrar

理由：
- 域名買完直接整合 Cloudflare，無需設定 Nameserver
- 成本價，長期持有最便宜
- 管理全部集中在 Cloudflare Dashboard
```

### 💰 預算有限，買多個域名

```
推薦：Porkbun > Namecheap > Dynadot

理由：
- Porkbun .com 續費最低（約 $10.73）
- 多個域名時差價積累明顯
- Porkbun 附免費 SSL 和 Email 轉寄，省去額外費用
```

### 🏢 企業使用，需要一站式服務

```
推薦：GoDaddy 或 Namecheap

理由：
- GoDaddy 提供 Hosting、Email、Website Builder
- 有中文客服（GoDaddy）
- 遇到問題有足夠的社群資源
- 預算充足時，服務整合更重要
```

### 🌍 需要歐洲或特殊 ccTLD

```
推薦：Gandi > Namecheap > Dynadot

理由：
- Gandi 支援最廣泛的歐洲 TLD（.fr、.de、.eu 等）
- 資料存於歐洲，符合 GDPR
- 附贈 2 個 Email 信箱
```

### 🇹🇼 台灣用戶，需要中文介面

```
推薦：Dynadot > GoDaddy

理由：
- Dynadot 有繁體中文介面
- 支援台灣常用付款方式
- GoDaddy 也有中文，但續費貴
```

### 🤖 開發者，需要 API 管理

```
推薦：Cloudflare Registrar > Porkbun > Namecheap

理由：
- Cloudflare API 最完整，文件清楚
- Porkbun API 簡潔易用
- 三家均有完整 REST API
```

### 🧪 長期練習用，不打算商用

```
推薦：Porkbun + .xyz 或 Porkbun + .dev

核心原則：
- 續費要夠便宜（會持有好幾年）
- TLD 不重要，選便宜的就好
- 搭配免費 Cloudflare，練習 DNS / HTTPS / 反向代理等所有技術

方案一：最省錢 → Porkbun + .xyz
  首年：~$1.73 USD
  續費：~$3.23 USD/年  ← 每年不到 NT$100
  適合：只是要有個域名指向某個 IP 或服務

方案二：練習 HTTPS 強制 → Porkbun + .dev
  首年 / 續費：~$10.73 USD/年
  特性：.dev 是 HSTS Preloaded，瀏覽器強制走 HTTPS
  適合：練習 SSL 設定、Nginx/Caddy reverse proxy、Let's Encrypt

方案三：最省心（不想管 Nameserver）→ Cloudflare Registrar + .com
  費用：~$9.77 USD/年（固定）
  特性：買完自動整合 Cloudflare，DNS / CDN / SSL 全部現成
  適合：想一次搞定所有設定，專注在練習本身而非工具設定
```

**TLD 練習用價格參考（Porkbun 續費，USD/年）**

| TLD | 續費 | 備註 |
|-----|------|------|
| `.xyz` | ~$3.23 | 最便宜，最常見的練習用 TLD |
| `.click` | ~$3.48 | 極便宜 |
| `.fun` | ~$3.98 | 便宜 |
| `.me` | ~$6.73 | 個人用途感強 |
| `.dev` | ~$10.73 | Google 擁有，強制 HTTPS，適合練 SSL |
| `.app` | ~$10.73 | 同 .dev，強制 HTTPS |
| `.com` | ~$10.73 | 最通用，練習真實場景 |

> **注意**：`.dev` 和 `.app` 屬於 HSTS Preloaded TLD，瀏覽器不允許 HTTP 連線，
> 一定要有有效的 SSL 憑證才能正常訪問，適合刻意練習 HTTPS 流程。
> 純粹想簡單掛個服務的話選 `.xyz` 即可。

# 常見陷阱與注意事項

### ⚠️ 首年低價陷阱

GoDaddy 等域名商常以超低首年價格（如 $0.99、$1.99）吸引購買，但**第二年續費價格大幅上漲**。

```
建議：
1. 購買前先查清楚「續費價格」，而非只看首年價
2. 短期測試可用 GoDaddy 首年促銷，到期前轉移到便宜的域名商
3. 長期使用直接選 Cloudflare 或 Porkbun
```

### ⚠️ WHOIS 隱私保護需付費

部分域名商（如 GoDaddy）的 WHOIS 隱私保護需要額外付費（每年 $9.99），等於實際費用更高。

```
建議：
選擇免費提供 WHOIS 隱私的域名商：
Namecheap / Cloudflare / Porkbun / Gandi / Dynadot
```

### ⚠️ 轉移鎖定期

ICANN 規定域名新購或轉入後 **60 天內不能轉移**。購買低價首年後，想轉移到別處至少要等 60 天。

### ⚠️ 自動續費

多數域名商預設開啟自動續費，信用卡到期可能導致扣款失敗而域名過期。

```
建議：
1. 設定 Email 提醒（提前 30、60、90 天）
2. 確保付款資訊有效
3. 重要域名建議購買 2 ~ 5 年
```

### ⚠️ Nameserver 設定後 DNS 管理位置改變

將 Nameserver 指向 Cloudflare 後，原域名商的 DNS 設定**全部失效**，必須在 Cloudflare 重新設定所有記錄。

```
建議：
轉換前先截圖或記錄原有 DNS 記錄，
Cloudflare 會自動掃描匯入，但需人工確認是否完整。
```
