# BlackHole(虛擬音訊裝置)

```
BlackHole 是一款免費且開源的 macOS 虛擬音訊驅動程式，允許應用程式之間相互傳遞音訊、零延遲。

2ch 版本提供 2 個音訊通道，適合擷取電腦系統聲音（瀏覽器、遊戲、音樂等）並傳送至錄音或直播軟體。
```

# 安裝

## Homebrew (MacOS)

```bash
brew install blackhole-2ch
```

## 官方網站下載

前往[官方網站](https://existential.audio/blackhole/)，填寫電子郵件以取得下載連結。

# 設定：錄製系統聲音（同時保留耳機/喇叭輸出）

## 步驟 1：建立多重輸出裝置

1. 開啟 macOS 內建的「音訊 MIDI 設定」應用程式。
2. 點擊左下角的「+」按鈕，選擇「建立多重輸出裝置」。
3. 在右側清單中，勾選平時使用的「外接耳機/內建揚聲器」以及「BlackHole 2ch」。
4. 勾選「主要裝置」旁的「漂移修正」，確保音訊同步。

## 步驟 2：更改系統音訊輸出

1. 點擊 macOS 螢幕右上角的「聲音」圖示（或前往系統設定 > 聲音）。
2. 將輸出裝置切換為剛建立的「多重輸出裝置」。

## 步驟 3：在錄音／直播軟體中擷取

1. 開啟錄音軟體（例如 OBS Studio、QuickTime Player）。
2. 將該軟體的「音訊輸入來源」或「麥克風來源」設定為「BlackHole 2ch」。
3. 開始錄製，即可擷取系統內部的所有聲音。
