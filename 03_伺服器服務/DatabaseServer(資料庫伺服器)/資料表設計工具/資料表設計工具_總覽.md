# 需求選擇建議

| 情境 | 推薦工具 | 理由 |
|---|---|---|
| MySQL 設計新資料表，需要 ER 圖 | **MySQL Workbench** | 官方工具，正逆向工程完整 |
| 開發初期快速討論 Schema | **dbdiagram.io** | 語法直覺，可分享連結，DBML 可納入版本控制 |
| 不想裝軟體、拖拉介面設計 | **DrawDB** | 瀏覽器即用，可匯出 SQL |
| 分析現有 MySQL / 多種 DB 結構 | **DBeaver** | 免費通用，直接從 DB 反產 ER 圖 |
| 分析現有 MongoDB 集合結構 | **MongoDB Compass** | 官方免費，Schema 取樣即可用 |
| 正式設計 MongoDB 文件模型 | **Hackolade** | 原生支援 BSON 型別、嵌入結構、匯出 Validator |
| 需要完整 MongoDB IDE + Schema | **Studio 3T** | 功能最全，Schema 只是其中一部分 |
