# 指令

## 查看所有服務（不論是否啟動）

`使用 systemctl（最常用）`

```sh
systemctl list-units --type=service --all
```

列出所有 systemd 管理的服務，包括：

active（正在運行）

inactive（已停止）

failed（啟動失敗）

`只看正在執行的服務`

```sh
systemctl list-units --type=service
```

## 顯示目前「哪些程式正在佔用埠號」

```sh
ss -tulnp
```

t：TCP

u：UDP

l：正在 listen（監聽）

n：顯示數字而非主機名

p：顯示對應的程式與 PID

## 只看 TCP（例如伺服器常見端口）

```sh
netstat -tulnp
```

如果出現「command not found」，可以先安裝：

```sh
apt install net-tools
```
