# Linux 工具 sar(監控Linux)

```
sar（System Activity Reporter, 系統活動情況報告）: 是用於監控Linux系統各個效能的優秀工具，包括：檔案的讀寫情況、系統呼叫的使用情況、磁碟I/O、CPU效率、記憶體使用狀況、程序活動及IPC有關的活動等
```
