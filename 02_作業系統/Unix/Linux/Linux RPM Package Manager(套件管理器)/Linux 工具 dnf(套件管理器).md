# Linux 工具 dnf(套件管理器)

```
RPM發行版的軟體套件管理器Yellowdog Updater, Modified（yum）的下一代版本。
DNF旨在解決部分yum已知的不足之處，包括糟糕的效能、高記憶體使用、緩慢的迭代拓撲排序。
```
