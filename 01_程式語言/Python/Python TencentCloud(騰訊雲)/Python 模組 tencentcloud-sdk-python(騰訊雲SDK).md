# 指令

```bash
# 腾讯云 Python SDK
pip install --upgrade tencentcloud-sdk-python

# 只安裝個別產品(不能與總包共同啟用)
pip install --upgrade tencentcloud-sdk-python-common tencentcloud-sdk-python-cvm

# 对象存储的 XML Python SDK
pip install -U cos-python-sdk-v5
```
