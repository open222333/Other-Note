# Python 模組 selenium(爬蟲)

```
特色

	能夠輕鬆與JavaScript的事件合作
	可以處理網頁的AJAX請求
	自動化操作網頁上的元素
	使用時機

當所要爬取的網頁有使用JavaScript / AJAX等動態載入資料的技術，或是具有登入驗證、搜尋機制的網頁，就能夠利用Selenium套件的網頁操作方法(Method)，來開發Python網頁爬蟲。

須根據瀏覽器安裝相對應的驅動(需對應瀏覽器版本)
```
