# Python 模組 pandas(強大資料處理與分析套件)

```
pandas 是 Python 的強大資料處理與分析套件。
它能讓你輕鬆處理表格、CSV、JSON、Excel、SQL 資料等。

常見用途：
用途	範例
讀取資料	pd.read_csv("data.csv")、pd.read_excel("file.xlsx")
資料整理	過濾、排序、群組、轉換、合併
匯出資料	df.to_csv()、df.to_excel()、df.to_json()
與 NumPy 結合	高速數值運算與統計分析
```
