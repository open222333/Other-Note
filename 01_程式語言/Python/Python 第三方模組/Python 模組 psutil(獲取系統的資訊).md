# Python 模組 psutil(獲取系統的資訊)

```
psutil可以獲取系統的很多資訊，包括CPU，記憶體，磁碟，網路，並且還可以檢視系統程序等資訊。psutil是一個跨平臺的庫，支援linux、windows、freebsd、OSX等多個作業系統平臺。
```
