# Python 模組 uvicorn(ASGI Web伺服器)

```
Uvicorn 是 Python 的 ASGI Web 服務器實現

WSGI 是 Web Server Gateway Interface 的縮寫，是一種協議，或是叫做規範。WSGI 定義了 web server 跟 python web application 之間溝通的規範。
```
