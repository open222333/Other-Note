# Python 模組 Pillow(影像處理)

Pillow類別(需安裝)
pip install pillow
ImageColor模組：
函數 getrgb()：將顏色符號或字串轉為元組，可使用英文名稱("red")、色彩數值(#00ff00)、rgb函數(rgb(0,255,0))、rgb函數(0%,0%,100%)
函數 getcolor(color,"mode")：
    參數"RGBA"：返回RGBA模組
    參數"RGB"：返回RGB模組

Image模組：
屬性 size：取得寬和高。width,height = 物件.size
屬性 filename：取得檔案名稱。
屬性 format：取得檔案格式。
屬性 format_description：取得檔案描述
方法 save()：儲存檔案
方法 new(mode,size,color=0)：建立新的影像物件。
    參數 mode：可以有多個設定，一般建議用"RGBA"(建立png檔案)或"RGB"(建立jpg檔案)。
    參數 size
方法 resize((width,height),image.BILINEAR)：調整影像大小。
方法 rotate()：旋轉角度
方法 transpose()：讓影像翻轉。
    參數 FLIP_LEFT_RIGHT：影像左右翻轉
    參數 FLIP_TOP_BOTTOM：影像上下翻轉
方法 getpixe((x,y))：取得影像某一位置像素(pixel) 的色彩
    參數(x,y)：元組，像素位置。
方法 putpixe((x,y),(r,g,b,a))：在影像某一位置填入色彩像素(pixel) 的色彩
    參數(x,y),(r,g,b,a)：元組，像素位置、色彩元組。
方法 crop((左,上,右,下))：裁切影像。
方法 paste(插入影像,(x,y))：(x,y)元組是插入位置。

ImageFilter模組：
方法 filter()：為圖片將上濾鏡。
    參數 BLUR：模糊
    參數 CONTOUR：輪廓
    參數 DETAIL：細節增強
    參數 EDGE_ENHANCE：邊緣增強
    參數 EDGE_ENHANCE_MORE：深度邊緣增強
    參數 EMBOSS：浮雕效果
    參數 FIND_EDGES：邊緣訊息
    參數 SMOOTH：平滑效果
    參數 SMOOTH_MORE：深度平滑效果
    參數 SHARPEN：銳利化效果

ImageDraw模組：
方法 point([(x1,y1),...(xn,yn),fill])：繪製點。
    參數 fill：設定顏色。
方法 ellipse((dleft,top,right,bottom),fill,outline)：繪製圓形。
    參數 (dleft,top,right,bottom)：包住圓或橢圓的左上角與右下角的座標。
    參數 fill：RGBA()或直接指定顏色。
    參數 outline：外框顏色。
方法 rectangle((dleft,top,right,bottom),fill,outline)：繪製圓形。
    參數 (dleft,top,right,bottom)：矩形左上角與右下角的座標。
    參數 fill：RGBA()或直接指定顏色。
    參數 outline：外框顏色。
方法 polygon([(x1,y1),...(xn,yn)],fill,outline)：繪製圓形。
    參數 (x,y)：多邊形左上角與右下角的座標。
    參數 fill：RGBA()或直接指定顏色。
    參數 outline：外框顏色。
方法 text((x,y),text,fill,font)：填寫文字。
    參數 text：想寫入的文字
    參數 font：需導入ImageFont模組，ImageFont.truetype()方法選用字型。˙˙

盒子元組(box tuple):(left,top,right,bottom)
    left:盒子左上角的x軸座標
    top:盒子左上角的y軸座標
    right:盒子右下角的x軸座標
    bottom:盒子右下角的y軸座標

QRCode模組(需安裝)
pip install qrcode
qrcode模組：
方法 make(Obj):建立QRcode物件。
    參數 Obj:資料。ex:網址
方法 QRCode(version,error_correction,box_size,border,image_factory,mask_pattern)
    參數 version:QR code版次 可設定1-40間
    參數 error_correction:設定從容錯率
        qrcode.constants.ERROR_CORRECT_L:7%
        qrcode.constants.ERROR_CORRECT_M:15%(預設)
        qrcode.constants.ERROR_CORRECT_Q:25%
        qrcode.constants.ERROR_CORRECT_H:30%
    參數 box_size:每個模塊的像素個數。
    參數 border:這是邊框區的厚度，預設是4。
    參數 image_factory:圖片格式，預設是PIL。
    參數 mask_pattern:參數是0-7，若省略會自行使用最適當方法。
