# 指令

```bash
# 安裝
apt-get install python3-venv

# 建立存放venv位置
mkdir /home/udoms/env/

# 建立venv
python3 -m venv /home/udoms/env/md

# 設置環境變數
source /home/udoms/env/md/bin/activate
```
