# API文檔 轉換 Markdown文檔

[postman-to-markdown](https://www.npmjs.com/package/postman-to-markdown?activeTab=readme)

```bash
# 安裝js套件
npm install -g postman-to-markdown

# 轉換
postman-to-markdown Target.postman_collection.json
```
