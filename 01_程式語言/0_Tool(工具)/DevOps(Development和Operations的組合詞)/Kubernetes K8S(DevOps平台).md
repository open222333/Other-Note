# Kubernetes K8S(DevOps平台)

```
Kubernetes是用於自動部署、擴充和管理「容器化應用程式」的開源系統。
該系統由Google設計並捐贈給Cloud Native Computing Foundation來使用。
它旨在提供「跨主機叢集的自動部署、擴充以及執行應用程式容器的平台」。
```
