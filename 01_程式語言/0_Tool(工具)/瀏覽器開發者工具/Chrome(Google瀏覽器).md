# 開發者工具

## 查看 SSL證書 資訊

1. Chrome瀏覽器在右上角點擊`⋮`，然後點擊`更多工具`，進入`開發者工具`。

2. 進入開發者工具後選擇`Security`。

3. 點擊`View Certificate`查看證書。
