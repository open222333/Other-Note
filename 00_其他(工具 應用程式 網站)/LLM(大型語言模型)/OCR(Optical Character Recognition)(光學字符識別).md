# OCR(Optical Character Recognition)(光學字符識別)

```
Optical Character Recognition

一種技術，用於將圖像中的文字轉換為可編輯的文本。
這項技術對於從掃描的文件、照片中的文字以及其他包含文字的圖像中提取數據非常有用。
OCR 系統通過識別圖像中的字符形狀，將其轉換為對應的文本數據。
```
