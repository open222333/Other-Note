# cdnjs(靜態資源CDN服務商)

```
cdnjs 是一個專注於 JavaScript、CSS 和其他前端庫的 CDN 服務商。
它提供了各種流行的前端庫和框架的 CDN 分發，包括 Vue.js、React、Bootstrap 等。
```
