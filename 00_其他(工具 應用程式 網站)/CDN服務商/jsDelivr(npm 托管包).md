# jsDelivr(npm 托管包)

```
jsDelivr 是一個開源的 CDN 服務商，提供了廣泛的 npm 托管包的快速、可靠的分發。
它支持 JavaScript、CSS、圖像等資源的分發，並且提供了版本管理、自動壓縮等功能。
```
