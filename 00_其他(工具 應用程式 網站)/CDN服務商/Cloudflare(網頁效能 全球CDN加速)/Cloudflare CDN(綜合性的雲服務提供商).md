# Cloudflare CDN(綜合性的雲服務提供商)

```
Cloudflare 是一個綜合性的雲服務提供商，其中包括 CDN 功能。
它提供了全球分發、智能緩存、安全防護等功能，並且可以與其他 Cloudflare 的服務集成使用。
```
