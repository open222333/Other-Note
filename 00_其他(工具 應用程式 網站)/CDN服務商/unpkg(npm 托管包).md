# unpkg(npm 托管包)

```
unpkg 是一個流行的開源 CDN 服務商，也提供了許多 npm 托管包的 CDN 分發。
它以簡單、直觀的方式提供了對各種 npm 包的訪問，並且支持通過版本號或標籤來引入特定版本的資源。
```
