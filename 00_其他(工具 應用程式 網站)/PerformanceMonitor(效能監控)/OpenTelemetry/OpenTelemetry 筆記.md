# OpenTelemetry 筆記

```
一個開源項目，旨在提供跨語言、跨平台的分散式系統觀察和監控解決方案。
它的目標是幫助開發者在分散式環境下實現更好的可觀察性，包括監控、追蹤和日誌等方面。
```
