# 簡單郵件傳送協定 SMTP(Simple Mail Transfer Protocol)

```
簡單郵件傳送協定（英語：Simple Mail Transfer Protocol，縮寫：SMTP）可用在傳送和接收電子郵件的資訊，但SMTP通常用作傳送電子郵件資訊，而不是接收。

SMTP 是用於發送電子郵件的主要協議。
它定義了如何從發送者的電子郵件客戶端或伺服器發送電子郵件到收件人的電子郵件伺服器。
```
