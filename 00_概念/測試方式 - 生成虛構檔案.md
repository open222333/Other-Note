# 測試

## 建立 sparse 檔案（快速，不真實寫入）

```sh
truncate -s 1G dummy_1GB.bin
```

## 建立真實 1GB（會寫入、較慢）

```sh
dd if=/dev/zero of=dummy_1GB.bin bs=4M count=256
```

if=/dev/zero	輸入來源為 /dev/zero，這是一個會產生連續零（0x00）的虛擬裝置。

of=dummy_10GB.bin	輸出檔案名稱，會在當前目錄建立這個檔案。

bs=4M	每次寫入的區塊大小為 4 MB。

count=256	總共寫入 256 個區塊。

計算檔案大小:

```
4 MB × 256 = 1024 MB = 1 GB
```
